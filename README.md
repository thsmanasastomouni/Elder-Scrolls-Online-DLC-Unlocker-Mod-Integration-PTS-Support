This tool grants you access to every DLC expansion pack while playing on the Elder Scrolls Online Public Test Server. You retain your ability to organize files through the Minion mod manager. ## 💻 System Requirements
Your computer must meet these basic standards to run the software correctly:

* Game installation: The Elder Scrolls Online Public Test Server must be installed on your local drive through the official game launcher.
* Storage: You need at least 50 megabytes of free space for the tool.
* Permissions: You need administrator access to your local Windows user account to allow the software to modify game configuration files.

## 🔧 Troubleshooting Common Issues
* **The game does not show the DLCs:** Restart the game client. Ensure you launched the Public Test Server version and not the main live server version.
* **The installer cannot find the game:** Use the manual browse button during the installation process. Point the tool to the folder where you installed the game, typically located under Zenimax Online/The Elder Scrolls Online PTS.
* **Mod conflicts:** If a specific addon causes issues, disable it within the game menu to isolate the problem. This tool functions at the base configuration level and does not interfere with standard user interface mods.
* **Security software flags the tool:** Some antivirus programs view third-party modifications as unknown software. If a block occurs, add an exclusion for the mod folder in your security software settings.
* **Update requirements:** Every time the Public Test Server receives a major game patch, download the latest version of this tool from the project page. Old versions may stop working when the game files change.

## 🧩 Compatibility Details
